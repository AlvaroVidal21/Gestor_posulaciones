# Gestor de Postulaciones

## Objetivo
Tener trazabilidad de las postulaciones laborales que estoy realizando para conseguir un puesto como practicante en el área de TI, permitiendo hacer seguimiento al estado de cada una y evitar olvidarlas.

## Problemas que resuelve
Evita que la información de mis postulaciones esté dispersa en diferentes medios, como hojas de cálculo, notas, correos o mensajes. Además, me permite tener mayor consciencia de mi progreso, del estado de cada proceso y de las acciones pendientes.

## Usuarios
- Yo, como postulante a prácticas preprofesionales en TI.

## Alcance
El sistema solo registra postulaciones laborales. No gestiona becas, cursos, certificaciones ni otros procesos personales.
