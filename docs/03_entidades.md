# Entidades

## Postulación

Representa una solicitud laboral realizada por el usuario para conseguir un puesto como practicante o junior en TI.

### Atributos
- id
- empresa
- puesto
- plataforma
- url_oferta (opcional)
- fecha_postulacion
- fecha_ultima_actualizacion
- estado: ["Postulado", "Vencido", "Rechazado"]
- notas (opcional)
