# Decisiones

## DEC-001
Se utilizará SQLite como base de datos debido a su simplicidad y porque el sistema tendrá un único usuario.

## DEC-002
La empresa será un atributo de la postulación y no una entidad independiente.

## DEC-003
El estado "Vencido" será calculado por el sistema y no almacenado en la base de datos.

## DEC-004
El dashboard mostrará métricas generales para facilitar el seguimiento del proceso de búsqueda laboral.
- Total de postulaciones
- Postulaciones activas
- Postulaciones rechazadas
- Postulaciones vencidas
- Postulaciones por plataforma
